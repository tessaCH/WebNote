If garbled messages show on Tomcat Server window, right click on top bar -> content -> current character coding page (eg. Big5) \conf -> logging.properties file open with an editor -> "java.util.logging.ConsoleHandler.encoding = UTF-8" -> change UTF-8 to Big5 3.use ctrl+C in Tomcat Server window to shutdown server, then restart again and you'll see the change applied
